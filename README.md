# Master-Painter 3-D Showcase + Paint-Your-Room

## Deploy on GitHub Pages (free)
1. Fork / upload this repo exactly as-is.  
2. Settings → Pages → Branch: `main` → Save.  
3. Site goes live at `https://YOUR_USERNAME.github.io/painter-3d-cms`

## Admin (media upload)
- Visit `https://YOUR_USERNAME.github.io/painter-3d-cms/admin`  
- Login with GitHub → drag images → Publish → auto rebuilds.

## Paint-Your-Room
- Click “Paint Your Room” on homepage → pick room → paint walls/ceiling → save PNG.
